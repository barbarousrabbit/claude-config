## uv — Primary Package Manager

uv is an extremely fast Python package and project manager written in Rust. It replaces pip, pip-tools, pipx, poetry, pyenv, twine, and virtualenv.

```bash
# Install uv (if not already installed)
curl -LsSf https://astral.sh/uv/install.sh | sh   # Linux/Mac
powershell -ExecutionPolicy Bypass -c "irm https://astral.sh/uv/install.ps1 | iex"  # Windows

# Project initialization
uv init myproject              # Create new project with pyproject.toml
uv init --lib myproject        # Create library project (src layout)

# Python version management (replaces pyenv)
uv python install 3.13         # Install Python 3.13
uv python install 3.14         # Install Python 3.14
uv python install 3.13-freethreading  # Install free-threaded build
uv python pin 3.13             # Pin version in .python-version

# Dependency management
uv add requests                # Add dependency
uv add --dev pytest ruff mypy  # Add dev dependencies
uv remove requests             # Remove dependency
uv lock                        # Update lock file
uv sync                        # Sync venv with lock file

# Running commands in the project venv
uv run pytest                  # Run pytest in project venv
uv run mypy src                # Run mypy in project venv
uv run ruff check src tests    # Lint
uv run ruff format src tests   # Format (replaces black)
uv run python -m myproject     # Run module

# pip interface (for scripts / non-project use)
uv pip install requests        # Install a package
uv pip install -e ".[dev]"     # Editable install with extras
uv pip compile requirements.in -o requirements.txt  # Lock deps
uv pip sync requirements.txt   # Sync venv to lockfile

# Building and publishing
uv build                       # Build sdist and wheel
uv publish                     # Publish to PyPI

# Tool management (replaces pipx)
uv tool install ruff           # Install CLI tool globally
uv tool run ruff check .       # Run tool without installing (like npx)
uvx ruff check .               # Shorthand for uv tool run
```

## Package __init__.py

```python
# src/myproject/__init__.py
"""MyProject - A Python package."""

from myproject.core import main_function, CoreClass
from myproject.utils import helper_function

__version__ = "0.1.0"
__all__ = ["main_function", "CoreClass", "helper_function"]

# Package-level configuration
import logging

logger = logging.getLogger(__name__)
logger.addHandler(logging.NullHandler())

# Python 3.14+: no need for `from __future__ import annotations`
# Forward references work out of the box with deferred evaluation (PEP 649)
```

## Version Management

```python
# src/myproject/__version__.py
__version__ = "0.1.0"

# src/myproject/__init__.py
from myproject.__version__ import __version__

# Read version in pyproject.toml (Python 3.11+ has tomllib built-in)
import tomllib
from pathlib import Path

def get_version() -> str:
    pyproject = Path(__file__).parent.parent / "pyproject.toml"
    with open(pyproject, "rb") as f:
        data = tomllib.load(f)
    return data["project"]["version"]
```

## Dependency Management Best Practices

```python
# Pin dependencies for applications — use uv.lock (auto-managed)
# uv.lock is cross-platform, deterministic, and auto-generated

# Use ranges for libraries in pyproject.toml
# requests>=2.32.0,<3.0.0
# pydantic>=2.10.0,<3.0.0

# Update dependencies
uv lock --upgrade              # Upgrade all
uv lock --upgrade-package requests  # Upgrade specific package
uv sync                        # Sync venv with lock file
```

## CI/CD Integration

```yaml
# .github/workflows/test.yml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ["3.13", "3.14"]

    steps:
    - uses: actions/checkout@v4

    - name: Install uv
      uses: astral-sh/setup-uv@v5
      with:
        python-version: ${{ matrix.python-version }}

    - name: Install dependencies
      run: uv sync --frozen

    - name: Run tests
      run: uv run pytest --cov --cov-report=xml

    - name: Type check
      run: uv run mypy src

    - name: Lint and format check
      run: |
        uv run ruff check src tests
        uv run ruff format --check src tests

    - name: Upload coverage
      uses: codecov/codecov-action@v4
```

## Pre-commit Hooks

```yaml
# .pre-commit-config.yaml
repos:
  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.8.0
    hooks:
      - id: ruff           # Lint
        args: [--fix, --exit-non-zero-on-fix]
      - id: ruff-format    # Format (replaces black)

  - repo: https://github.com/pre-commit/mirrors-mypy
    rev: v1.13.0
    hooks:
      - id: mypy
        additional_dependencies: [types-requests]
```

```bash
# Install pre-commit
uv tool install pre-commit     # or: uv pip install pre-commit
pre-commit install

# Run manually
pre-commit run --all-files
```
